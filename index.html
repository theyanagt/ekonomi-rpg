⚠️ ANOMALI EKONOMI
🚨 Masalah Balance yang Ditemukan
1.
Rock beli 1.000, jual 850 — spread sangat sempit. Pemain bisa farm rock dari nambang dan jual nyaris break even. Tidak ada insentif menarik beli rock di shop.
2.
Nambang tidak butuh Pickaxe — secara desain harusnya pickaxe meningkatkan hasil mining, tapi di file nambang.js tidak ada pengecekan pickaxe sama sekali. Bug atau disengaja?
3.
Berburu reward sangat besar — money 10k-360k per sesi, EXP 3k-75k per sesi (60 menit CD). Bisa 576k - 8.640.000 money/hari dari berburu saja dengan gear Netherite.
4.
Judi player range 0-75, bot 0-100 — win rate player matematika hanya ~36%. House edge sangat tinggi, tidak sustainable.
5.
Bibit berkebon minimum 500 setiap jenis — butuh 2.500 bibit total × 5 rupiah = 12.500 per sesi tapi reward buah sangat kecil jika dijual.
6.
Nambang tanpa cooldown di gagal — saat gua runtuh (gagal), lastfishing/nambang tetap dicatat. Sudah benar.
✅ Saran Perbaikan Balance
CRAFTTambahkan cek pickaxe di nambang.js. Pickaxe tier tinggi = yield rock/iron lebih banyak.
JUDIUbah rollPlayer ke 0-90 (bukan 0-75) agar win rate ~47.5%. Atau tambah pity system.
BERBURUKurangi money reward berburu 30-40%. Ganti dengan lebih banyak hewan item daripada direct money.
BERKEBONKurangi minimum bibit dari 500 ke 50 per jenis. Atau buat bibit tidak habis (perkebunan permanen).
ROCKTurunkan harga beli rock ke 500 atau naikkan harga jual ke 1.200 agar lebih worth tambang lalu jual.
SHOPTambahkan komisi 5-10% saat sell item untuk sink uang dari ekonomi.

🔍 KESIMPULAN BALANCE
✅ BAIKSistem musim & jam di shop/pasar/transfer sudah ada dan cukup kompleks. Harga dinamis tiap jam, weekend naik, musim berpengaruh. Ini bagian terbaik sistem.
⚠️ INFLASIMoney masuk terlalu besar vs money keluar. Ngewe 30-150k per 10 menit = bisa 4-21 juta/hari. Tidak ada money sink yang cukup kuat.
🔧 FIXTambahkan pajak jual di shop/pasar (sink 5-10%). Tambahkan biaya repair gear secara otomatis. Tambahkan sistem tax progresif saat berburu berulang.
⚠️ JUDIWin rate judi solo hanya ~36% (player 0-75 vs bot 0-100). Terlalu merugikan player. Pemain akan berhenti judi setelah rugi berkali-kali.
✅ BAIKSistem gear tier 1-5 dengan durability bagus. Memberi tujuan jangka menengah. Multiplier berburu dari x1 sampai x4+ memberi insentif upgrade.
✅ BAIKStreak daily (7 hari → uncommon crate) adalah engagement loop yang efektif untuk player aktif.

🌟 REKOMENDASI: Fitur yang Cocok Ditambah Musim & Jam
⏰ FITUR COCOK DITAMBAH JAM
🎣 MANCING
Jam Sibuk (subuh 04-06, sore 16-18): Ikan aktif menggigit → +20-30% jumlah hasil
Jam Sepi (10-14 siang): Ikan malas → -20% hasil
Weekend: Banyak pemancing amatir → ikan lebih sulit +10% chance gagal
🏹 BERBURU
Jam Sibuk (fajar 05-07, senja 17-19): Hewan aktif → +25% yield
Jam Sepi (12-15 tengah hari): Hewan bersembunyi → -20% yield
Malam (22-04): Hewan nokturnal berbeda, bisa bonus khusus
⛏️ NAMBANG
Jam Normal (08-17): Normal
Jam Malam (22-05): Gua lebih berbahaya → +5% chance gagal tapi +diamond bonus 5%
Weekend: Banyak penambang → rock/iron lebih sedikit (overcrowded)
🛵 OJEK
Jam Sibuk (07-09, 17-19): Banyak orderan → +50% money
Jam Sepi (02-06 dini hari): Orderan sepi → -30% money
Weekend: Orderan antar tempat hiburan → +30% money
🦹 MALING
Jam Malam (21-04): Lebih gelap → -10% chance ketahuan
Jam Siang (10-15): Terlalu terang → +15% chance ketahuan
Weekend: Warung ramai → +20% chance ketahuan
🗑️ MULUNG
Pagi (05-07): Orang buang sampah → +30% sampah
Malam (20-23): Pasar tutup, banyak sisa → +20% sampah
Weekend: Lebih banyak sampah dari pesta → +40% botol & kaleng
🌱 BERKEBON
Pagi (05-09): Waktu terbaik berkebun → hasil buah +20%

function getJamMultiplier() {
    const hour = new Date().getHours()
    const day = new Date().getDay()
    const isWeekend = day === 0 || day === 6

    // Jam sibuk berbeda per fitur — sesuaikan!
    const jamSibuk = [4,5,6,16,17,18]       // pagi subuh & sore (contoh mancing/berburu)
    const jamSepi  = [10,11,12,13,14]        // tengah hari

    let mult = 1.0
    let label = '☀️ Jam Normal'
    if (jamSibuk.includes(hour)) { mult = 1.3; label = '🔥 Jam Sibuk' }
    else if (jamSepi.includes(hour)) { mult = 0.8; label = '😴 Jam Sepi' }
    if (isWeekend) { mult *= 1.1; label += ' (Weekend)' }
    return { mult, label, isWeekend }
}

function getMusimMultiplier(kategori = 'ikan') {
    const month = new Date().getMonth() + 1
    const musim = {
        '❄️ Hujan':  { ikan: 1.3, hewan: 0.9, wood: 1.3, bibit: 0.8 },
        '🌸 Semi':   { ikan: 1.1, hewan: 1.2, wood: 1.0, bibit: 1.4 },
        '☀️ Kemarau':{ ikan: 0.85,hewan: 1.4, wood: 0.9, bibit: 1.2 },
        '🍂 Gugur':  { ikan: 1.0, hewan: 1.1, wood: 1.2, bibit: 1.0 },
    }
    let nama, data
    if ([12,1,2].includes(month))  { nama='❄️ Hujan';   data=musim['❄️ Hujan'] }
    if ([3,4,5].includes(month))   { nama='🌸 Semi';    data=musim['🌸 Semi'] }
    if ([6,7,8].includes(month))   { nama='☀️ Kemarau'; data=musim['☀️ Kemarau'] }
    if ([9,10,11].includes(month)) { nama='🍂 Gugur';   data=musim['🍂 Gugur'] }
    return { mult: data[kategori] || 1.0, nama }
}
