⚖️ LOGIKA BALANCING & ANTI-EXPLOIT
⚠️
ANTI-INFLASI: Tanpa balancing yang ketat, kombinasi jam sibuk + weekend + musim peak bisa menghasilkan reward 3–5× normal, menyebabkan hiperinflasi ekonomi RPG dalam hitungan hari.
🛡️ PRINSIP BALANCING
1. MULTIPLIER CAP (Batas Maksimum)
Semua multiplier dikombinasikan dengan batas keras: totalMult ≤ 3.5×. Jika kombinasi melebihi ini, reward di-cap di 2× base. Mencegah exploit jam sibuk + weekend + musim peak.
2. SEEDED RANDOM (Stabilitas Harga)
Harga shop digenerate dengan seeded random berbasis waktu (Math.sin(seed+offset)*10000). Ini memastikan harga konsisten selama 1 jam penuh, tidak berubah setiap detik, mencegah price manipulation.
3. COOLDOWN IMMUTABLE
Cooldown tidak bisa di-bypass dengan cara apapun. Berburu (60 min), Nambang (30 min), Ojek (5 min) — durasi dikunci ketat. Mencegah flood spam untuk income tidak terbatas.
4. FAIL CHANCE (Distribusi Risiko)
Setiap aktivitas punya chance gagal (10–35%). Gagal = tidak ada reward + kehilangan resource. Balances high-reward activities agar tidak terlalu mudah dieksploitasi.
5. ZERO-SUM PVP
Aktivitas PVP (merampok, membunuh, judi) bersifat zero-sum: uang tidak diciptakan, hanya dipindahkan. Ini tidak mempengaruhi total money supply dalam ekonomi.

// IMPLEMENTASI WAKTU (wajib ditambahkan)
function getTimeMult() {
  const h = new Date().getHours()
  if (h >= 6 && h < 12 || h >= 20) return 1.35
  if (h >= 12 && h < 20)            return 1.00
  return 0.70 // jam sepi 00–05
}
// IMPLEMENTASI HARI
function getDayMult() {
  const day = new Date().getDay()
  const isWeekend = day===0 || day===6
  // Sudah ada di rpg-shop.js!
  return isWeekend ? seededRand(1.10,1.45) : seededRand(0.85,1.15)
}
// PATCH UNTUK rpg-hourly.js, rpg-berburu.js, dll:
const timeMult = getTimeMult()
const dayMult = getDayMult()
const capMult = Math.min(timeMult * dayMult, 2.0)

// Apply ke reward:
let money = isPrems
  ? rand(4000,15000) * capMult
  : rand(800,7000) * capMult
let exp = isPrems
  ? rand(8000,18000) * (capMult*0.85)
  : rand(3000,9000) * (capMult*0.85)
