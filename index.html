🕐 ADAPTASI BERDASARKAN JAM
🔴 JAM SIBUK — 18:00–23:00
Jam paling ramai. Banyak member aktif. Optimal untuk aktivitas sosial & PvP. Reward rate tertinggi, tapi saingan juga banyak.
  • ⏰
    Hourly PASSIVE
    Wajib klaim tiap jam. EXP + Money. Cooldown 1 jam.
  • 📋
    Daily REWARD
    Klaim jika belum. Bonus Streak + item crate. Cooldown 24 jam.
  • 🛵
    Ojek REWARD
    Banyak "penumpang" aktif. Money + EXP. Cooldown 5 menit — spam saat sibuk!
  • ⚔️
    Hunter / Berburu REWARD
    Monster spawn lebih sering. Reward x1.5 saat malam. Butuh armor+sword.
  • 🗺️
    Adventure REWARD
    Crate drops lebih berharga. Cooldown 1 jam. Butuh armor 80 HP.
  • 🦹
    Merampok / Membunuh RISKY
    Banyak target kaya aktif. Tapi risiko gagal 35%. Lakukan setelah stamina penuh.
  • 🎲
    Judi / JudiPvP RISKY
    Banyak lawan. Tapi hati-hati modal habis.
  • 💋
    Ngelonte PREMIUM
    Reward besar. Cooldown 12 jam. Lakukan sebelum tidur. Money 150k–500k.

🕐 Deteksi Jam Otomatis
Tambahkan helper function di awal setiap handler: 
function getTimeZone() {
  const jam = new Date().getHours()
  if (jam >= 18 || jam < 0)  return 'SIBUK'
  if (jam >= 10 && jam < 18) return 'NORMAL'
  return 'SEPI' // 00:00–09:59
}

function isWeekend() {
  const hari = new Date().getDay()
  return hari === 0 || hari === 6
}

function getMoneyMultiplier() {
  let mult = 1
  if (getTimeZone() === 'SIBUK')  mult *= 1.5
  if (getTimeZone() === 'SEPI')   mult *= 0.8
  if (isWeekend())                mult *= 1.3
  return mult
}
📅 Deteksi Musim & Event
Sistem event berbasis tanggal: 
function getCurrentEvent() {
  const now = new Date()
  const bulan = now.getMonth() + 1
  const tgl = now.getDate()

  // Ramadan (perkiraan, atau gunakan API)
  if (bulan === 3 || bulan === 4) return 'RAMADAN'
  
  // Lebaran
  if (bulan === 4 && tgl <= 7)   return 'LEBARAN'

  // Tahun Baru
  if (bulan === 1 && tgl <= 3)   return 'TAHUNBARU'

  // Kemerdekaan
  if (bulan === 8 && tgl === 17) return 'MERDEKA'

  // Musim Kemarau (April–September)
  if (bulan >= 4 && bulan <= 9)  return 'KEMARAU'

  // Musim Hujan (Oktober–Maret)
  return 'HUJAN'
}

function getEventMultiplier(event) {
  const mult = {
    RAMADAN: 2.0, LEBARAN: 2.5,
    TAHUNBARU: 2.0, MERDEKA: 1.8,
    KEMARAU: 1.3, HUJAN: 1.1
  }
  return mult[event] || 1.0
}
